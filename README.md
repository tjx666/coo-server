coo-server 是 coo 项目的后端服务器，基于我总结的 [koa2 RESTful 开发模板](https://github.com/tjx666/koa-restful-boilerplate) 编写 RESTful API，使用 socket.io 进行服务端推送。

## 本地开发

### 获取项目

```bash
# 克隆本项目
git clone https://github.com/tjx666/coo-server.git
```

### 安装依赖

```bash
# 安装依赖，推荐使用 yarn
yarn
```

### 启动项目

```bash
yarn start
```

### 本地测试

```bash
yarn test
```
